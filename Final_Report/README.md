# Final Report – Customer Churn Analysis

This folder contains the final report deliverables for the ACS WILDA Customer Churn Analysis project.

## Overview
The final report summarises the end-to-end data analytics process, including data preparation, clustering analysis, predictive modelling, and business insights. The objective of this project is to identify key factors influencing customer churn and provide data-driven recommendations to support customer retention strategies.

## Files Included

- **Final_Report_Team2.docx**  
  The complete final report combining all sections of the project.

- **Final_Summary.docx**  
  A concise summary of key findings, insights, and outcomes.

- **Final_Factors.docx**  
  Analysis of the main factors contributing to customer churn.

- **Final_Recommendations.docx**  
  Data-driven recommendations to reduce churn and improve retention.

- **Final_Limitations.docx**  
  Discussion of project limitations and potential areas for future improvement.

## Key Highlights
- Identification of important churn drivers through data analysis  
- Use of clustering techniques to segment customers  
- Development of an Artificial Neural Network (ANN) model for churn prediction  
- Actionable business recommendations based on analytical findings
