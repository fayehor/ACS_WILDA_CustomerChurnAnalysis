# ACS_WILDA_CustomerChurnAnalysis

ACS WILDA Project – Customer Churn Analysis (Team 2)

## Project Overview
This project focuses on analysing customer churn behaviour in a telecommunications dataset. The goal is to identify key factors influencing churn and develop data-driven strategies to improve customer retention.

The project follows a complete data analytics pipeline, including data preparation, clustering analysis, and predictive modelling.

## Project Structure

- **Data_Preparation/**
  - Data cleaning, preprocessing, and feature engineering
  - Includes handling missing values, encoding, and train-test split

- **Clustering_Analysis/**
  - Customer segmentation using clustering techniques (e.g., K-Means)
  - Includes baseline and tuned clustering models

- **Predictive_Modeling/**
  - Artificial Neural Network (ANN) model for churn prediction
  - Includes baseline and tuned models, evaluation, and results

- **Final_Report/**
  - Final report documents, including:
    - Full report
    - Summary
    - Key factors
    - Recommendations
    - Limitations

## Key Techniques Used
- Data preprocessing and feature engineering  
- Clustering analysis (customer segmentation)  
- Artificial Neural Networks (ANN) for prediction  
- Model evaluation and performance tuning  

## Key Outcomes
- Identified important factors influencing customer churn  
- Segmented customers into meaningful groups  
- Developed a predictive model to estimate churn probability  
- Provided actionable business recommendations
